NFC CONTACT CARD TEMPLATE

UPDATED FEATURE:
- The Venmo QR image is now clickable.
- Clicking the Venmo QR opens the Venmo link from data.js.

HOW TO USE:

1. Replace images in:
   assets/images/

   profile.png   -> your profile picture
   venmo_qr.png  -> your Venmo QR code

2. Replace icons in:
   assets/icons/

3. Edit ALL links and text inside:
   data.js

4. Replace files:
   assets/files/resume.pdf
   assets/files/contact.vcf

5. Open index.html in your browser to test.

6. Upload the whole folder to GitHub Pages or your web host.

IMPORTANT VENMO NOTE:
- Edit the Venmo link in data.js
- Keep imageClickable: true
- Clicking the QR image will open that Venmo URL
