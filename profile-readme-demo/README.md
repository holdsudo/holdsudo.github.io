# holdsudo // happy bird demo

```text
BLACK SKY RENDERED
BIRD LOADED
PIPES APPROACHING
```

This is a GitHub-native README game demo.

It has no JavaScript.
It works with plain Markdown links and section jumps.

## how to play

Click one of the commands below.

[ press space ](#press-space)
[ drop low ](#drop-low)
[ panic ](#panic)

---

## press space

```text
score: 1

      __
  ___(o )>

          | |
          | |
```

The bird climbs cleanly through the first gap.

Choose:

[ climb again ](#climb-again)
[ level out ](#level-out)

---

## drop low

```text
score: 0

      __
  ___(x )>
      ""

      | |
  ____| |
```

You dropped too early and clipped the lower pipe.

[ retry ](#how-to-play)

---

## panic

```text
score: 0

SYSTEM OVERRIDE DENIED
BIRD CONFUSED
```

The bird forgot how to flap.

[ retry ](#how-to-play)

---

## climb again

```text
score: 2

           __
      ___(o )>

   | |
   | |
```

Too much lift. Ceiling proximity warning.

Choose:

[ correct course ](#level-out)
[ keep climbing ](#keep-climbing)

---

## level out

```text
score: 3

        __
   ___(o )>      | |
                 | |
```

Perfect line. The bird is happy.

Choose:

[ thread final gap ](#thread-final-gap)
[ show README vault ](#readme-vault)

---

## keep climbing

```text
score: 2

   BONK
   ___(x )>
```

Ceiling collision.

[ retry ](#how-to-play)
